Virtual Devices Pro adds virtual devices to Homey that you can use as building blocks in your Flows. A virtual device holds a value — a switch state, a number, a text, a sensor reading — without any physical hardware behind it.

Use them as state holders, dummy sensors, scene toggles or bridges between Flows.

Supported device types:

- Virtual Switch — a simple on/off switch
- Virtual Button — a button with on/off/toggle Flow cards
- Virtual Number — a numeric value with optional unit, decimals and range
- Virtual Text — a text value with equals/contains conditions
- Virtual Temperature — a temperature sensor (°C)
- Virtual Power — a power sensor (W)
- Virtual Battery — a battery level (%)
- Virtual Humidity — a humidity sensor (%)
- Virtual Luminance — a light sensor (lux)
- Virtual Pressure — an air pressure sensor (mbar)
- Virtual CO2 — a CO2 sensor (ppm)
- Virtual Contact — an open/closed contact sensor
- Virtual Motion — a motion sensor
- Virtual Dimmer — a dimmable light with on/off and dim level
- Virtual Energy Meter — a cumulative energy meter (kWh) with set and add actions

Every device comes with Flow cards to read, compare and set its value. Devices with standard capabilities also work with Homey's built-in Flow cards, Insights and Energy.

When adding a device you can pick a name and choose from 27 built-in icons.

All values are stored locally on your Homey. The app makes no external connections, collects no data and requests no permissions.
